Place .txt files here with one survey URL or DOI per line.

Use the "Import TXT" button (or press 'i') in the TUI to batch-import them.
Lines starting with # are treated as comments and ignored.
Empty lines are ignored.

Example file (surveys.txt):
    # My TCC surveys
    https://ieeexplore.ieee.org/document/10123456
    https://ieeexplore.ieee.org/document/10654321
    10.1109/ACCESS.2024.1234567
